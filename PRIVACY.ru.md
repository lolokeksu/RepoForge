# Политика приватности

[English](PRIVACY.md) | [Русский](PRIVACY.ru.md)

RepoForge работает полностью офлайн.

Приложение не собирает, не загружает, не продаёт и не передаёт пользовательские данные.

RepoForge не использует:

- доступ к интернету
- аккаунты
- аналитику
- телеметрию
- рекламные SDK
- облачное хранилище
- удалённые серверы

Вся обработка проектов выполняется локально на устройстве.

Доступ к папкам выполняется через Android Storage Access Framework.

RepoForge создаёт ZIP-архивы только из папок, выбранных пользователем.
