# RepoForge

[English](README.md) | [Русский](README.ru.md)

**RepoForge** — компактное Android-приложение для создания чистых ZIP-архивов исходного кода проекта.

Приложение предназначено для подготовки проектов к AI-аудиту, проверке перед публикацией на GitHub и безопасному локальному бэкапу без build-мусора, временных файлов, APK-артефактов, локальных путей, ключей подписи и распространённых секретных файлов.

## Возможности

- Создание чистых ZIP-архивов из выбранной папки проекта
- Сканирование исходников перед упаковкой
- Отображение добавленных файлов, пропущенных файлов, размера исходников и итогового размера
- AI readiness check
- Проверка на распространённые секретные файлы и файлы подписи
- Три режима архивации:
  - AI Audit
  - GitHub Clean
  - Backup Safe
- Генерация Markdown-отчёта
- История созданных архивов
- Отправка последнего ZIP через системное меню
- Открытие папки результата
- Копирование пути к архиву
- Копирование краткой сводки
- Полностью офлайн-работа
- Root не требуется
- Разрешение на интернет не требуется
- Без аналитики
- Без телеметрии

## Режимы архивации

### AI Audit

Лучший режим для отправки проекта в Claude, ChatGPT или другой AI-инструмент для аудита кода.

Этот режим удаляет build-артефакты, Gradle-кэш, APK-файлы, локальную конфигурацию, IDE-файлы, архивы, логи и распространённые секретные файлы.

### GitHub Clean

Лучший режим для подготовки исходников перед публикацией или ревью репозитория.

Этот режим сохраняет обычные исходные файлы, но удаляет сгенерированные build-файлы, APK-артефакты, кэши, логи и локальную конфигурацию.

### Backup Safe

Лучший режим для локального бэкапа проекта.

Этот режим сохраняет больше проектных файлов, чем AI Audit, но всё равно исключает ключи подписи, секреты, `local.properties`, APK-артефакты, временные файлы и кэш-папки.

## Что RepoForge исключает

RepoForge исключает распространённые ненужные или чувствительные файлы:

- `build/`
- `.gradle/`
- `.idea/`
- `.vscode/`
- `local.properties`
- `*.apk`
- `*.aab`
- `*.ap_`
- `*.idsig`
- `*.zip`
- `*.7z`
- `*.tar`
- `*.tar.gz`
- `*.tgz`
- `*.log`
- `*.tmp`
- `*.bak`
- `*.jks`
- `*.keystore`
- `*.pem`
- `*.key`
- `*.p12`
- `*.pfx`
- `*.crt`
- `*.cer`
- `.env`
- `.env.local`
- `secrets.properties`
- `google-services.json`
- `service-account.json`

## Результат

Созданные архивы сохраняются здесь:

```text
Downloads/RepoForge/
```

Пример результата:

```text
GPD_repoforge_ai_audit_2026-06-23_18-30-00.zip
GPD_repoforge_report_ai_audit_2026-06-23_18-30-00.md
```

## Приватность

RepoForge работает локально на устройстве.

Приложение не:

- запрашивает доступ к интернету
- загружает проекты на серверы
- использует аккаунты
- собирает аналитику
- отправляет телеметрию
- требует root-доступ

Доступ к папкам выполняется через Android Storage Access Framework.

## Требования

- Android 10 или новее
- Root не требуется
- Интернет не требуется
