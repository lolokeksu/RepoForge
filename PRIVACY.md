# Privacy Policy

[English](PRIVACY.md) | [Русский](PRIVACY.ru.md)

RepoForge works fully offline.

The app does not collect, upload, sell, or transmit user data.

RepoForge does not use:

- internet access
- accounts
- analytics
- telemetry
- advertising SDKs
- cloud storage
- remote servers

All project processing happens locally on the device.

Folder access is handled through Android Storage Access Framework.

RepoForge creates ZIP archives only from folders selected by the user.
